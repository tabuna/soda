Три слоя защиты

Всё, что нужно для чистого и читаемого кода

📏

Structural

Максимальный размер классов, методов, аргументов, зависимостей и глубина неймспейсов.

🌀

Complexity

Цикломатическая сложность, вложенность условий и количество возвратов.

🌬️

Breathing

Читаемость имён, визуальное «дыхание» и комфорт восприятия кода.

Два простых режима

1
Анализ метрик проекта
php soda analyse src
2
Полная проверка качества
php soda quality src

Установка за 30 секунд

Composer пока не поддерживается. Используйте git clone. 
$ git clone https://github.com/tabuna/soda.git
cd soda
composer install
php soda init
